# Contribution Guidelines

Thank you for your interest in contributing to the Phone Popup Detector project!

## How to Contribute

1. **Clone the repository**: Use `git clone https://github.com/wan-dl/phone-popup-detector.git` to get a local copy of the repository.

2. **Create a branch**: Itâ€™s best practice to create a new branch for your contribution. You can do this using `git checkout -b your-feature-branch`.

3. **Make your changes**: Implement your feature, fix bugs, or improve documentation.

4. **Commit your changes**: Make sure to write clear and concise commit messages that describe your changes.

5. **Push your changes**: Use `git push origin your-feature-branch` to push your changes back to GitHub.

6. **Create a Pull Request**: Go to the repository on GitHub, and you should see an option to create a Pull Request (PR) for your branch.

## Code of Conduct

By participating in this project, you agree to uphold our code of conduct and contribute in a respectful manner.

## Issues

If you encounter any issues or have suggestions, please use the GitHub issue tracker to report them. 

We appreciate your contributions, and we look forward to collaborating with you!
